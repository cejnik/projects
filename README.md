# Projects Portfolio

This folder contains my practice and portfolio backend projects built while learning Django and Symfony.

The goal of these projects is to gradually improve in backend development, database design, authentication, permissions, validation, and API work.

## Projects

### Symfony Team Training Planner

A Symfony learning project focused on authentication, role-based access, training management, attendance, and capacity rules.

Main topics:
- authentication and protected routes
- coach and participant roles
- training CRUD
- ownership checks for edit and delete
- attendance join and leave flow
- capacity validation
- flash messages
- basic functional controller testing

Repository folder:
- `Symfony_Team_Training_Planner`

### Symfony Room Booking

A Symfony learning project focused on room booking, authentication, ownership, and basic REST API endpoints.

Main topics:
- Doctrine entities and relations
- authentication and authorization
- room and reservation CRUD
- ownership checks
- REST API with GET, POST, PATCH, and DELETE
- API testing in Insomnia

Repository folder:
- `Symfony_room_booking`

### Django Task Manager

A Django portfolio project focused on projects, tasks, memberships, filtering, and permissions.

Main topics:
- model relationships
- project membership logic
- task assignment
- filtering by status and priority
- authentication and permissions
- automated tests

Repository folder:
- `Django_task_manager`

### Django Cinema Reservation System

A Django portfolio project focused on movie reservations, screenings, and seat selection.

Main topics:
- reservation flow
- screening selection
- seat selection
- authentication
- model relationships
- automated tests

Repository folder:
- `Django_cinema`
